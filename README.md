## 🔖 專案導覽

### ✨ 提交規範

- 🎉 init：專案初始化
- ✨ feat：新增功能（feature）
- 🐞 fix：修復bug
- 📃 docs：文件修改
- 🌈 style：程式碼樣式調整，不影響程式碼邏輯
- ✅ test：測試相關的改動
- 🔨 refactor：程式碼重構
- 🔧 chore：建置過程或輔助工具的變動

### 🎯 相容環境

- 現代瀏覽器（Chrome >= 64, Edge >= 79, Firefox >= 78, Safari >= 12）

### 💻 本地調試

```bash
$ git https://github.com/BlackishGreen33/Codetest.git
$ cd Codetest
$ pnpm install
$ pnpm run dev
```
